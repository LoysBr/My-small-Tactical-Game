# Tactical Game - Unity Experimental Project

A small-scale tactical game built in Unity to explore modern game development practices, Agentic AI workflows, and maintain a collection of code examples I can easily refer back to or reuse.

## 🎯 Project Goals

### 🎮 Game Development
- Explore new Unity features (rendering tools, Cinemachine system...)
- Implement and review key programming patterns through practical application
- Conduct performance testing in C# and Unity
- Deploy a web playable Demo

### 📚 Learning Objectives 
Refresh my memory with things I haven't used for a long time or explore new tools and techniques.

- **AI Agents development Workflows:**
  - Using Claude inside Unity + MCP 
  - Test with specialized Agents Team: Programmer, UI/3D Integrator, QA Tester

- **Mathematics & Geometry**
  - Review notions of Velocity, Acceleration for Movement
  - Vector and Matrices operations, Space transformation 

- **Apply Design Patterns:**
  - MVP, Command, Observer/Events, Singleton, Component, State Machine
  - Service Locator, Pooling, Spatial Partitioning (QuadTree), Behavior Tree
  - ...

- **Use integrated Unity Network System**

- **Test Modern Backend solutions**

---

## 🧠 Game Design

### Core Mechanics
- **Turn-based tactical combat** with 2 controllable characters (Mage & Fighter)
- **Single map** with procedurally distributed enemies
- **Objective:** Eliminate all enemies on the battlefield

### UI/UX Elements
- Ability list for current character
- Controllable character roster
- Turn order display

## Current Status

### ✅ Implemented
- Map with density-based enemy distribution
- Animated, controllable player character
- Camera controls 
- Simple Coding Agent Workflow 

### 🔄 In Development

### 📋 Planned
